# java-lotto-precourse
## 기능 목록

1) 로또 구매
    - 사용자로부터 구매할 금액을 입력받는다.
    - 금액만큼 랜덤으로 로또 번호를 생성한다.
    - 생성된 로또 번호를 출력한다.


2) 당첨 번호 입력
    - 당첨번호를 쉼표로 구분해서 입력받는다.
    - 쉼표를 기준으로 번호를 분리하여 정답 배열로 만든다.
    - 별도의 보너스 번호를 입력 받는다. 

3) 당첨 여부 확인기능 구현 
    - 정답과 생성한 로또 번호를 비교하여, 몇등에 당첨되었는지를 구분한다.
    - 등수 별 당첨된 로또 갯수를 저장한다.

4) 당첨 내역 출력
    - 당첨 갯수에 맞는 포멧으로 결과를 출력한다.
    - 입력받은 금액 대비 수익률을 계산하여 출력한다.

5) 잘못된 입력에 대한 예외처리
    1. 로또 번호가 1이하거나 45 초과인 경우
    2. 로또 번호에 중복된 숫자가 있는 경우
    2. 구매 금액이 1000원 미만인 경우
    3. 구매 금액이 1000원 단위로 나눠떨어지지 않는 경우
    4. 정답 로또 번호에 중복된 숫자가 존재하는 경우 
    5. 숫자가 아닌 입력이 들어오는 경우
